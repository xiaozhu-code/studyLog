## starcoin安装及节点配置

> **starcoin安装**

从 github 的 [releases](https://github.com/starcoinorg/starcoin/releases) 页面下载适合自己操作系统的二进制，放置到任意文件夹并配置环境变量(path)。



> **配置节点并加入网络**

**加入主网络**

在命令行中直接运行 `starcoin` 命令，默认会在前台启动一个节点并尝试加入主网。



**启动本地网络节点**

想加入其他的网络，可以用 `-n` 或 `--net` 选项来指定，比如启动一个本地的 `dev` 网络节点。

在启动交互式控制台时通过 `-d` 选项指定数据目录，则不会在进程结束后将其删除：

```
starcoin -n dev -d . console
```

`.` 表示当前目录，也可以使用一个绝对路径来指定存放数据的目录。
